Ruby -> Interpreter Languange

1. Run Ruby From Command Line
   $ ruby [options] [.] [programfile] [arguments ...]
   
   Show Manual Pages :
   $ ruby -h
   $ ruby --help

2. Interactive Ruby
   $ irb
   $ irb --noprompt
   $ irb -h
   $ irb --help
   $ irb -v
   $ irb --version 
   $ irb[.rb] [options] [programfile] [arguments ...]

3. Possible Shebang Lines
   #! /usr/bin/ruby
   #! /usr/local/bin/ruby 

   #! /usr/bin/env ruby    -> Run Command in a Program

4. File Permission
   $ chmod 777 hello.rb
   $ ./hello.rb
