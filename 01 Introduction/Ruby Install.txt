ENVIRONMENT SETUP

Install Ruby
1. $sudo apt install ruby-full
2. $sudo apt install ruby-build
3. $sudo apt install ruby-bundler

Install mruby
1. $sudo apt install mruby

Install GTK, GTK3, ..,
1. $sudo gem install gtk3

Install Tk 
1. $sudo apt install tk         (Windows Shell) 
2. $sudo apt install tk-dev     (Development Files)
3. $sudo apt install tk8.5      (Ubuntu 18.04 or under)
4. $sudo apt install tk8.5-dev  (Ubuntu 18.04 or under)

Install Tcl
1. $sudo apt install tcl        (Windows Shell)
2. $sudo apt install tcl-dev    (Development Files)
3. $sudo apt install tcl8.5     (Ubuntu 18.04 or under)
4. $sudo apt install tcl8.5-dev (Ubuntu 18.04 or under)

Install Rake
1. $sudo gem install rake

Install Sublime Text
1. $ sudo snap install sublime-text --classic














REPOSITORY
/etc/apt/source.list
